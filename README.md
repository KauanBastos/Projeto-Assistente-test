# Projeto Assistente
# Assistente Virtual Magalu  Aplicativo de linha de comando para auxiliar tarefas de operação e pós‑login, com comandos de ajuda, hora/data, calculadora, tutor de login (abrir páginas e orientações) e fluxos de consolidação/consulta. Inclui build para executável Windows e instalador Inno Setup com ícone e wizard personalizados.  ## Principais Recursos - Ajuda com lista de comandos - Hora e data atual - Calculadora segura suportando parênteses e operadores aritméticos - Tutor de login com abertura de URLs e dicas (sinclog) - Logins frequentes: cadastro/consulta por mesa e senhas padrão - Pós‑login: fluxos de consolidação e consultas - Build do executável (.exe) e instalador Windows (Inno Setup) - Ícone e imagens do wizard gerados por script  ## Arquitetura - Entrada principal: `main.py`   - Instancia skills e inicia REPL; aceita entrada por argumentos de linha de comando. - Núcleo: `assistant/core.py`   - Roteia o texto para a primeira skill cujo `matches` retorna True; loop de execução com “sair/exit/quit”. - Skills:   - Ajuda: `assistant/skills/help.py`   - Hora/Data: `assistant/skills/time_skill.py`   - Calculadora: `assistant/skills/calc_skill.py`   - Logins frequentes: `assistant/skills/frequent_logins.py`   - Tutor de login: `assistant/skills/login_tutor.py`   - Pós‑login: `assistant/skills/post_login_tutor.py`  ## Comandos Suportados - `ajuda | help | comandos | ?` - `hora | horas | data` - `calc <expressão> | calcular <expressão> | quanto é <expressão>` - `login | login abrir sinclog | login 2fa | login esqueci senha | login erro` - `logins listar | logins set <mesa> <login> | login mesa <mesa>` - `senhas listar | senhas add <senha...> | senhas set <senha...> | senhas limpar` - `menu | consolidar | consolidar entregas | consolidar volumes | múltiplos destinos | nova consolidação | gaylord | sacas | consultar itens | consultas | consultar volumes | consultar destinos | NRO etiquetas | NRO nota fiscal`  ## Execução (Código‑fonte) 1. Requisitos: Python 3.12, pip 2. Instalar dependências:    ```bash    cd "C:\Users\User\Documents\trae_projects\assistente virtual"    py -3 -m pip install --user -r requirements.txt    ``` 3. Rodar:    ```bash    py -3 main.py    ```  ## Build do Executável e Instalador - Script PowerShell: `scripts/build.ps1`   - Instala dependências, gera ícone, injeta metadados de versão (Company, PublisherURL, Copyright)   - Gera .exe via PyInstaller e opcionalmente compila instalador via Inno Setup   - Logs: `dist\build.log` (PyInstaller), `dist\installer\iscc.log` (Inno Setup) - Script .bat opcional: `scripts/build_exe.bat`  ### Gerar .exe ```bash powershell -ExecutionPolicy Bypass -File .\scripts\build.ps1 ``` Saída: `dist\assistente-virtual-magalu.exe`  ### Gerar .exe + instalador ```bash powershell -ExecutionPolicy Bypass -File .\scripts\build.ps1 -WithInstaller ``` Saída: `dist\installer\Assistente Virtual Magalu-<versão>-Setup.exe`  ## Instalador (Inno Setup) - Script: `installer/assistente-virtual-magalu.iss` - Ícone do setup: `assets/assistant.ico` - Texto “antes de instalar”: `installer/INFO_BEFORE.txt` - Imagens do wizard: `installer/wizard_image.png`, `installer/wizard_small.png` - Metadados de versão e marca:   - CompanyName: Assistente Virtual Magalu   - Publisher URL: https://www.magalu.com   - Direitos autorais: © Magalu  ## Ícones e Artes - Gerador de ícone estático: `scripts/generate_static_icon.py`   - Saídas: `assets/assistant_icon.png`, `assets/assistant.ico` - Gerador de imagens do wizard: `scripts/generate_wizard_images.py`  ## Versão - Arquivo `VERSION` centraliza a versão (ex.: 1.0.0) e é usado no .exe e no instalador.
